So you got a new macro pad, huh?

You’re probably going to want to change the keys to something useful to you.

You could make them do basically anything!

  • Jumping between applications? Sure!
  • Writing out long commands you repeat a lot? That too!
  • A dedicated keyboard for your favorite slack emojis? You get the point now

Go to the Via webapp to get started

Click the keys in the layout to specify which key you want to change.

Then click the key you want to change it to

There are lots of types of keys you can assign. Theres the basic keycodes you’d find on a normal keyboard in the normal section. The rest are as follows:

  • Media: Media keys like play, pause, etc
  • Macros: Trigger a macro
  • Layers: Keycodes to switch to different layers in different ways. More info here
  • Special: A bunch of other things like shifted punctuation, mouse control, etc

How to configure macros

First head to the macros screen

Then you can either record a sequence of keypresses (with the movie slate icon) or type in a sequence of key codes (the html tag looking button)

Now you can go back to the keymap view (the little book icon back where the macros button is) and assign the corresponding macro key.
